Restful API 的设计就是通过 HTTP 的方法来表示 CRUD 的相关操作。除了使用 GET 和 POST 方法外，还会用到其他 HTTP 方法，比如 PUT,DELETE,HEAD 等，通过不同的 HTTP 方法来表示不同的含义。
